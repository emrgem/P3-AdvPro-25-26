CSV Format Requirements

Your CSV file should have a header row with column names. We support the following columns:

CSV Column Maps To Required?
Series_Title or title Movie Title Required
Released_Year or year Release Year Optional
Genre or genre Genre Optional
Director or director Director Optional
IMDB_Rating or rating Rating (0-10) Optional
Overview or description Description Optional
Poster_Link or poster_url Poster Image URL Optional
Sample CSV Format
Series_Title,Released_Year,Genre,Director,IMDB_Rating,Overview,Poster_Link The Shawshank Redemption,1994,Drama,Frank Darabont,9.3,"Two imprisoned men...",https://example.com/poster.jpg The Godfather,1972,Crime,Francis Ford Coppola,9.2,"An organized crime...",https://example.com/poster2.jpg
Tip: The IMDB Top 1000 dataset works perfectly with this importer!
Import Tips
  • Duplicates: Movies with the same title will be imported (no duplicate checking by default)
  • Missing data: Rows without a title will be skipped
  • Special characters: Quotes in descriptions are handled automatically
  • Large files: Import happens in one batch commit for speed
