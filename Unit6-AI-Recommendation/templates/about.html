About CineMatch

Our Mission

CineMatch helps movie lovers discover their next favorite film through personalized recommendations and a curated collection of cinema's finest works.

Built With Modern Tech

CineMatch is a student project built with:

  • Flask - Python web framework
  • Bootstrap 5 - Responsive design
  • SQLAlchemy - Database management
  • TMDB API - Real movie data and posters
  • AI Integration - Personalized recommendations (coming soon!)

Learning Journey

This application is part of our CTE Web Development course. Throughout this course, we've built:

  • Database design and SQLAlchemy ORM
  • CRUD operations (Create, Read, Update, Delete)
  • User authentication and authorization
  • External API integration (TMDB)
  • Search, filtering, and pagination
